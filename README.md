## Getting started
1. [Download](https://duke79.github.io/Jarvis/) & install.
2. Launch.
3. Keyboard shortcuts -
   1. `Ctrl+Shift+Space` to show
   2. `Esc` to hide.
   3. `Enter` to pick a history item
