## chaakar

![Icon](./Chaakar%20Icon.svg)

Augments developer.

### Run
```
npm run dev
```

### Publish
1. Generate a GitHub [token](https://github.com/settings/tokens/new)
2. Set env
```
export GH_TOKEN=<generated_token>
```
3. Build and publish
```
npm run package:dist
```
